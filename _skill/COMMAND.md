# 📋 Command Reference

ดัชนีรวมคำสั่งทั้งหมดของโปรเจค

---

## 🎯 Claude Code Commands

### `/fast`
**Description**: เปิด Fast Mode เพื่อให้ Claude ตอบเร็วขึ้น  
**Usage**: `/fast`

---

### `/model [model_name]`
**Description**: เปลี่ยนโมเดล Claude ที่ใช้  
**Usage**: 
```
/model haiku      # เร็ว, ประหยัด
/model sonnet     # สมดุล
/model opus       # ทรงพลัง, ดีที่สุด
```

---

### `/help`
**Description**: ดูความช่วยเหลือและรายการคำสั่ง  
**Usage**: `/help`

---

### `/config`
**Description**: ตั้งค่า Claude Code  
**Usage**: `/config`

---

## 🔄 Project Commands

### `update skill`
**Description**: อัปเดต SKILL.md ให้ทันสมัยตามโค้ดปัจจุบัน  
**Usage**: `update skill`  
**สิ่งที่อัปเดต**:
- ✅ Pages/Routes ใหม่
- ✅ API endpoints ใหม่
- ✅ Components ใหม่
- ✅ Custom hooks ใหม่
- ✅ Dependencies ใหม่
